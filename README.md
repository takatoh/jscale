# jscale

地震の加速度記録から計測震度と震度階を求めます。

## Install

``` get install github.com/takatoh/jscale```

## Usage

地震の加速度記録は CSV 形式で、3成分必要です。
たとえば次のように：

```
Time,NS,EW,UD
0.00,-0.03,0.01,-0.34
0.01,0.18,0.42,0.05
0.02,-0.17,-0.25,0.34
0.03,-0.55,0.41,0
0.04,0.31,0.76,0.62
0.05,-0.72,0.1,0.37
0.06,-0.04,0,-0.3
0.07,-0.13,0.46,-0.09
0.08,-0.69,-0.22,0.36
（後略）
```
加速度記録のファイルを example.csv とすると、次のように実行します。

``` jscale example.csv```
